# CS471 Demo CI

This is a sample repository to demo Continuous Integration (CI) using GitHub Actions.

This repository is using as a starting point the [CS471-Assignments-Builder](https://github.com/BoiseState/CS471-Assignments-Builder) repository, which uses:
- the [Gradle](https://gradle.org/) build tool
- the [gson](https://github.com/google/gson) external library/dependency

and is able to parse [JSON](https://www.json.org/) files.
